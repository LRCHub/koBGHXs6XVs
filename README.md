# LOT SPiRiTS – 二日月

> **歌詞登録ステータス：プレーン歌詞のみ**

```
夜の空に君が吐いた
ため息混じり微かな声
あの頃の僕達には
何も見えなかった
どうにか繋ぎとめた
細い糸は切れてしまって
見上げた、月が綺麗と笑った
君を解れなかった
今さら“君に会いたい”とか
“忘れられない”とか
覚めない夢の中
せめてこのまま
君の全てを奪い去って
今すぐここから連れ去って
揺らいだ想いは宙を舞って
終わりを告げるかのように
消えていく
“あなたが隣に居なくたって
わたしは変わらず生きてく”って
笑ってごまかすその癖は
最後も君を隠してしまう
“わたしが隣に居なくたって
あなたは笑っていられる”って
離したその手を掴むように
伸ばした空まであと少し
さよなら
欠けてく月のような
消えてく君の影
見失わないように
手を伸ばしても
夜が全てを奪い去って
光のひとつも見えなくて
君はどこにもいなくなって
明けない夜に迷い込んでいく
```
